# 🏥 Doctor Appointment API (Django REST Framework)

A RESTful API for managing doctors, patients, and appointments.

## 🚀 Features

- User registration (Doctor or Patient)
- Authentication with Token
- Doctor profiles
- Patient profiles
- Appointment management
- Ownership-based access control
- Validation for appointment conflicts

---

## 🧱 Tech Stack

- Python
- Django
- Django REST Framework
- SQLite (default)
